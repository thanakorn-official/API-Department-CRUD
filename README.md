### 📝 รายละเอียดโปรเจกต์ (Project Overview)
ระบบหลังบ้านสำหรับบริหารจัดการข้อมูลแผนก (**API-Department-CRUD**) พัฒนาขึ้นเพื่อเป็นระบบ RESTful API ที่รองรับการทำงานพื้นฐานครบวงจรแบบ **CRUD (Create, Read, Update, Delete)** มีการออกแบบโครงสร้างโค้ดที่สะอาด แยกสัดส่วนชัดเจน และจัดการข้อมูลผ่านระบบฐานข้อมูลอย่างเป็นระบบ ⚡

### 🛠️ เทคโนโลยีที่ใช้ (Tech Stack)
* **Backend Runtime:** Node.js 🟢
* **Database:** SQL Server / Relational Database 🗄️
* **API Testing Tool:** Postman / Thunder Client 📡
